# 🍪 Cookie Clicker Bot 🍪

A Python automation bot built with **Selenium** that plays the Cookie Clicker game automatically. The bot continuously clicks the cookie and purchases the most expensive unlocked upgrade every few seconds.

## Features

* Automatically clicks the main cookie
* Detects unlocked upgrades
* Buys the most expensive available upgrade
* Handles language selection automatically
* Uses Selenium for browser automation

## Tech Stack

* **Python**
* **Selenium WebDriver**
* **ChromeDriver**


## Run the Project

Run the Python file:

```bash
python main.py
```

The bot will:

1. Open Cookie Clicker in Chrome
2. Select English language
3. Continuously click the cookie
4. Buy the best unlocked upgrades
5. Print total cookies after ~16 seconds and close

## Project Purpose

This project was created to practice:

* Browser automation with Selenium
* Web element interaction
* Timing and automation logic
* Python scripting

## Requirements

* Python 3.x
* Google Chrome installed
* ChromeDriver (compatible with your Chrome version)
